# SM83 (Sharp LR35902)

Custom CPU in the Game Boy. Often called "GBZ80" but it's neither a Z80 nor an 8080.

## Relation to Z80

**Has:** 8-bit registers A, F, B, C, D, E, H, L. 16-bit SP, PC. Register pairs AF, BC, DE, HL. Flags Z, N, H, C. HALT, DI, EI. Relative jumps. CB prefix for bit operations.

**Doesn't have:** IX, IY. Shadow registers (AF', BC', DE', HL'). I/O instructions (IN, OUT). Block transfer (LDIR, LDDR). ED prefix. R register.

**Unique:** SWAP (CB 3x). STOP. Different flag behaviour on some operations.

## Timing

- Every instruction takes 1-6 M-cycles (4-24 T-cycles)
- 1 M-cycle = 4 T-cycles = 1 bus operation
- M1 (first M-cycle) always fetches the opcode from [PC]
- Subsequent M-cycles perform reads, writes, or internal operations

## Registers

| 8-bit | 16-bit pair | Notes |
|-------|-------------|-------|
| A, F  | AF          | F: flags, lower 4 bits always 0 |
| B, C  | BC          | General purpose |
| D, E  | DE          | General purpose |
| H, L  | HL          | General purpose / memory pointer |
| -     | SP          | Stack pointer |
| -     | PC          | Program counter |

## Flags (F register)

| Bit | Flag | Name |
|-----|------|------|
| 7   | Z    | Zero |
| 6   | N    | Subtract (BCD) |
| 5   | H    | Half-carry (BCD) |
| 4   | C    | Carry |
| 3-0 | -    | Always 0 |

## Internal Registers

The SM83 has internal latches not visible to software, used to hold intermediate values during multi-M-cycle instructions. We model these as `z` (low byte) and `w` (high byte), matching the Z80's WZ convention.

## PUSH/POP Register Pair Encoding

PUSH/POP use a different pair table from LD rr:

| Bits 5-4 | LD rr | PUSH/POP |
|----------|-------|----------|
| 00       | BC    | BC       |
| 01       | DE    | DE       |
| 10       | HL    | HL       |
| 11       | SP    | AF       |

POP AF masks the lower 4 bits of F to zero.

## Conditional Instructions

Condition encoding in bits 4-3 of the opcode:

| Code | Condition | Flag test    |
|------|-----------|--------------|
| 00   | NZ        | Z flag clear |
| 01   | Z         | Z flag set   |
| 10   | NC        | C flag clear |
| 11   | C         | C flag set   |

JR cc uses 2 M-cycles when not taken, 3 when taken (internal cycle for branch).
